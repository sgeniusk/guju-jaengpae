# 핵심 루프 정본 — 삼국지: 구주쟁패 (Nine Kings 정합)

전투·런 루프의 확정 정본이다. 채팅으로 흘러온 설계를 여기서 고정한다. 피처는 이 문서에 정합해야 한다. 확정 전 항목만 `(미정)`.

## 한 줄
군주 오토배틀러 로그라이크. **성을 지키며 오픈필드에서 적 군세를 막고, 골드로 덱을 키운다.** Nine Kings 벤치마크.

## 전장 (feat-013 done)
- **오픈필드 2D**. 배치 3×3은 *시작 진형*. 전투 시작하면 양쪽 군세가 들판으로 나와 만나 난전.
- 2D 최근접 타겟(컬럼 격리 없음), 원거리는 옆까지 커버. 스킬·병종 상성·상태(도발/약화) 적용.

## 성(城) — 방어 목표 (feat-014)
- **맨 안쪽 1칸 고정**. 보드 = 3×3 유닛 배치(9칸) + 성 1칸 = 10칸.
- 성은 높은 HP, 정적, 적의 표적. **적 군세 전멸 = 승, 성 파괴 = 패** (유닛 전멸이 아니라 성 파괴가 패배 조건).
- 적은 군세만(성 없음). 비대칭 방어. *(양쪽 성 대칭은 후속 검토.)*

## 핵심 루프 — 손패·보드·경제 (feat-015, NK식)
- **손패(Hand) 3장** — 항상 3장 보유.
- **보드(Board) = 영속 배치** — 카드를 보드 블록(타일)에 놓으면 런 동안 영속. 매 전투 그 배치로 군세가 스폰돼 싸운다(전투 결과와 무관하게 배치 유지). 블록 한정(예 3×3=9 + 성), 런 동안 채워진다.
- **카드 플레이 = 보드에 배치** — 손패 1장 → 보드 블록. 손패가 준다.
- **전리 보상** — 전투 승리 시 후보 3장 중 1장(패배시킨 **상대 진영 카드**). 받으면 손패 4장 → **3장이 되게 한 장 처리**(보드 배치 또는 우물 버리기).
- **우물(Well) 버리기** — 놓을 블록이 없거나 안 쓸 카드는 우물에 버려 **+10 골드**.
- **상점** — 4스테이지마다(StageCadence). 모은 골드로 카드 자유 구매. 구매분도 손패 초과 시 보드 배치 또는 우물. (feat-015d)
- **골드** — 전투·우물에서 모아 상점에서 소비.
> 긴장 — 손패 3 한계 + 보드 블록 한정. 좋은 카드는 배치, 자리 없으면 우물에 팔아 골드. 상대 진영 카드를 뺏어 군세를 키운다.
> 의존 — "상대 진영 카드" 보상은 6국 콘텐츠 필요. v1은 현 카드풀로, 진영 보상은 6국 때.

## 카드 종류
- 군주(君主)·장수(將帥)·병종(兵種) — 유닛.
- **배치 역할(전열/후열)은 카드 속성** — 장수=앞, 궁병/노병=뒤, 보병/기병=앞·중간. 카드 데이터의 역할 필드가 진형 위치를 정한다.
- **건물(建物)** — 타일에 짓는 정적 카드. 오라(주변 +공격력 등)·소환·경제. (feat-016)
- 계략(計略)·보패(寶貝) — 효과·유물. (후속)

## 전투 연출·조작
- **관전형 오토배틀** — 전투 시작하면 양 군세가 우르르 몰려 자동 교전, 사용자는 구경(확정).
- **대규모 떼** — NK는 수백 유닛이 느슨한 열로 진격해 덩어리로 충돌. 우리는 소규모로 프로토타입, **대형 정밀도는 후순위**("다른 문제").
- **영웅(장수) 조작 (후속 feat)** — 자동 전투 위에 미세 컨트롤 한 겹. 마우스/터치로 영웅의 표적(어느 적을 칠지) 지정. 관전+개입 하이브리드.

## 메타 — 선형 스테이지 사다리 + 캐이던스 (feat-019)
- **선형 스테이지 사다리** — stage 1→2→3… 끝없이 escalation. stage 번호가 적 강도·이벤트를 결정. 분기 노드맵(feat-007/008) 폐기.
- **스테이지 캐이던스** — 순수 로직 `StageCadence`가 stage 번호로 그 판 이벤트를 계산. 데이터 주도 간격.

| 이벤트 | 간격 | 발동 |
|---|---|---|
| 카드 드래프트 3중1 | 매 스테이지 | 전부 (기본) |
| 상점(골드 구매) | 4 | 4·8·12… (feat-015d) |
| 강적=보스 | 5 | 5·10·15… |
| 땅 확장(보드 +1랭크, 상한 18칸) | 5 | 5·10·15… 보스 보상 (feat-020) |
| 왕의 칙령(전역 perk 3중1) | 3 | 3·6·9·12… (feat-021) |

- **난이도** — 적 hp·공격이 stage에 비례 스케일.
- **왕의 칙령(王命)** — 3스테이지마다 전역 런 혜택 perk 3중1. 보드 카드가 아니라 런 전체 수정자, 누적. 예 — 지원 건물 효과 2배·전 유닛 공격력 +10%·골드 획득 +50%·성 HP +20%. 간격 3(knob, 3,9,15도 가능). 건물 관련 칙령은 건물(feat-016) 후. (feat-021)
- **9세력 = 9명의 왕**. 적은 황건적·마계 등 타 세력 군세(九州 worldview). 8명의 적 왕을 꺾으면 정복(6국 콘텐츠 때).

## 게임 모드 (향후 메타 — 구상)
사용자 구상 3모드 — **시나리오·자유·멀티**. 현재 런 = 사실상 자유모드(선형 스테이지 사다리, 끝없는 escalation).
- **자유모드** — 현재 루프. 배경 테마 기본 = 평원 진영.
- **시나리오모드** — 시나리오별 스테이지·적·**배경**이 바뀜(적벽·낙양·마계 침식 등). 배경은 단일 고정이 아니라 스테이지/시나리오 키로 선택하는 **교체형 테마 슬롯**.
- **멀티모드** — 후순위.
> v0.5는 모드 선택 UI·시나리오 데이터·멀티를 만들지 않는다. 단, 배경 테마(feat-025)·스테이지(StageCadence)를 **모드 키로 분기 가능하게** 설계해 둔다. 정본 [render-architecture.md](render-architecture.md) 배경 테마 시스템 참조.
