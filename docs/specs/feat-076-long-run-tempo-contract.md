# feat-076 장기런 전투 템포 계약

## 목표
- 중후반 전투가 25~35초 이상 늘어져 관전 피로가 생기는 문제를 자동 검증으로 잡는다.
- 보스전은 존재감을 유지하되, 한 전투가 오래 끌려 게임 템포를 죽이지 않게 한다.
- 초반 군세 밀도와 성 선점, 손패 3장 중 1장, 장기런 최종 승리 흐름은 유지한다.

## 범위
- 장기런 스모크에 전투 시간 상한과 군주별 평균 시간 상한을 추가한다.
- 보스 encounter의 체력만 좁게 조정해 유비, 조조, 손권 장기런이 시간 예산 안에서 끝나도록 한다.
- 시간 예산 helper를 단위 테스트로 검증한다.

## 비범위
- 새 카드, 새 군주, 새 스테이지 추가
- 전투 AI, 스킬, 보상, 저장 구조 변경
- UI 배치/아트 리워크

## 수용 기준
- 일반/정예/중간 보스 교전은 24초 이하로 끝난다.
- 최종 보스는 28초 이하로 끝난다.
- 각 군주의 장기런 평균 교전 시간은 18초 이하로 유지된다.
- 장기런 스모크가 위 조건을 어기면 실패한다.
- `./init.sh`가 통과한다.
