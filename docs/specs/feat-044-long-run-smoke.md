# feat-044 — 장기런 자동 스모크

## 목표
첫 5스테이지 MVP 메트릭을 넘어, 최종 보스까지 런 흐름이 끊기지 않는지 자동으로 검증한다.

## 범위
- 유비 기준 결정적 선택으로 stage 1부터 15까지 진행한다.
- 전투·보스·칙령·상점·사건·보드 확장 노드를 모두 통과한다.
- 각 전투 결과, 경과 시간, 보드 장수, 보드 레벨 합, 손패/드로우 수를 compact line으로 출력한다.
- `init.sh`에 장기런 스모크를 연결한다.

## 비범위
- 세 군주 전체 장기런 밸런스.
- 실제 화면 클릭 자동화.
- 난이도 완전 보정.

## 검증
- `tools/long_run_smoke.gd` 단독 실행.
- `./init.sh` 전체 green.
