# feat-061 전투 결과 복귀 안내

## 목표
전투 결과 화면에서 플레이어가 현재 런을 계속하는 버튼과 현재 런을 끝내고 새 런으로 돌아가는 버튼을 혼동하지 않게 한다.

## 범위
- 결과 오버레이에 `런 계속` 또는 `런 종료` 성격의 짧은 안내를 표시한다.
- 일반 승리 보상 뒤 `다음 스테이지로` 버튼 tooltip은 현재 런을 유지한 채 런맵으로 복귀한다고 말한다.
- `군주 선택으로 새 런` 버튼 tooltip은 현재 런을 초기화하거나 완료된 런 기록을 유지한 채 새 런을 시작한다고 말한다.
- 문구 계산은 `BattleOutcomeGuide` 순수 helper로 분리해 테스트한다.

## 비범위
- 전투 결과 판정, 프로필 저장, 보상 후보, 다음 stage advance 동작은 변경하지 않는다.
- 새 결과 화면 레이아웃이나 추가 모달은 이번 피처에 포함하지 않는다.

## 검증
- 단위 테스트가 패배, 최종 승리, 일반 승리의 안내 문구와 tooltip을 검증한다.
- 결과 화면 smoke가 패배/최종 승리 화면에서 종료 안내와 새 런 tooltip을 확인한다.
- UI smoke가 일반 승리 보상 후 현재 런 유지 tooltip과 새 런 포기 tooltip을 확인한다.
- `./init.sh` 전체 검증을 통과한다.
