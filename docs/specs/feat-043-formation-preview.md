# feat-043 — 배치 전술 미리보기

## 목표
전술 보너스가 실제 적용되기 전, 플레이어가 빈 타일을 보고 배치 의사결정을 할 수 있게 한다.

## 범위
- 손패 유닛 카드 선택 중 빈 타일마다 임시 보드를 계산한다.
- 해당 칸에 배치할 때 전술 보너스가 생기면 타일 라벨에 `지휘 +10%` 같은 짧은 미리보기를 표시한다.
- 미리보기 tooltip은 어떤 카드가 어떤 전술 보너스를 받는지 설명한다.

## 비범위
- 건물/계략 카드 미리보기.
- AI 추천이나 자동 배치.
- 새 전술 규칙 추가.

## 검증
- `FormationTactics.preview_for_unit()` 순수 helper를 단위 테스트한다.
- 기존 배치·전투 스모크와 `./init.sh` 전체 green을 유지한다.
