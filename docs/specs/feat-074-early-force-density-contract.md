# feat-074 초반 군세 밀도 계약 강화

## 목표
- 첫 전투부터 병종 카드가 충분한 병사 분대로 보이고, 장수도 혼자 큰 스프라이트가 아니라 호위병과 함께 지휘 축으로 읽힌다.
- 첫 5스테이지 자동 플레이테스트가 매 교전 최소 군세 밀도와 더 타이트한 시간 예산을 검증한다.
- 카드 데이터 정본과 전투 승패 구조는 바꾸지 않고, 기존 `SquadProfile`과 `PlaytestMetrics` 계약만 좁게 강화한다.

## 범위
- 병종 Lv.1 기본 분대 수를 올리고 레벨업 성장은 Lv.3 기준 18명 cap에 자연스럽게 닿게 조정한다.
- 장수 Lv.1 호위병 수를 올려 초반 장수 카드도 단독 1대1처럼 보이지 않게 한다.
- `PlaytestMetrics.first_five_ok()`가 매 교전 최소 아군 군세, 최소 전체 군세, 피크 군세, 초반 시간 예산을 검증한다.
- UI tooltip과 단위 테스트를 새 분대 수 계약에 맞춘다.

## 비범위
- 새 카드, 새 faction/nation id, 새 Resource 필드 추가
- 적 파도 구성 변경
- 보상, 저장, 맵 흐름 변경

## 수용 기준
- 병종 기본 분대는 UI와 `SquadProfile`에서 12명 이상으로 안내된다.
- 장수 Lv.1은 호위 7명 이상으로 시작한다.
- 첫 5스테이지 smoke는 매 교전 아군 visible soldiers 12명 이상, 전체 visible soldiers 30명 이상, 피크 아군 군세 30명 이상을 요구한다.
- `./init.sh`가 통과한다.
