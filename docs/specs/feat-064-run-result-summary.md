# feat-064 — 장기런 결과 요약 UX

## 목표
최종 패배와 최종 승리 화면에서 런이 그냥 끊기지 않고, 플레이어가 방금 쌓은 군세와 기록을 한눈에 읽게 한다.

## 배경
현재 결과 화면은 run_complete 여부와 프로필 최고 기록은 알려 주지만, 실제 런의 구성 성과는 보이지 않는다. 장기런의 재미를 만들려면 “이번 런에서 어디까지 갔고, 어떤 군세를 키웠는지”가 최종 화면에 남아야 한다.

## 범위
- `RunState`와 battle outcome을 입력으로 받는 순수 요약 helper 추가
- 최종 패배/최종 승리 결과 오버레이에 런 결산 블록 표시
- 전투 결과 스모크가 결산 문구를 검증

## 비범위
- 프로필 저장 스키마 변경
- 새 보상/해금 규칙
- 비최종 일반 승리의 전리품 흐름 변경

## 수용 기준
- 최종 패배 화면에 `런 결산 — 패배`, 스테이지, 점수, 군세, 최고 레벨 요약이 보인다.
- 최종 승리 화면에 `런 결산 — 승리`, 스테이지, 점수, 군세, 최고 레벨 요약이 보인다.
- 결산 tooltip에 칙령, 보패, 손패, 드로우 더미, 골드가 포함된다.
- 순수 unit test와 `tools/battle_result_smoke.gd`, `./init.sh`가 통과한다.
