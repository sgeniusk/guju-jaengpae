# feat-077 - 전투 첫 화면 지면/깊이 재보정

## 문제
배치 화면의 9칸 보드가 배경 위에 떠 있는 물체처럼 보이고, 교전 시작 때 유닛이 필드 뒤나 별도 레이어에서 나타나는 인상이 남아 있다. 직전 보정으로 격자 충돌은 줄었지만, 사람 눈이 읽는 "같은 바닥 위에 있다"는 단서가 부족하다.

## 목표
- 배치 보드 뒤에 넓은 전장 바닥 밴드를 렌더해 9칸이 지면 위에 놓인 판처럼 보이게 한다.
- 세 레인의 진군 바닥선을 배경 레이어에 남겨 성, 아군, 적이 같은 지면 축을 공유하도록 만든다.
- 교전 시작 후 격자는 계속 숨기되, 바닥 밴드와 레인은 유지해 유닛이 빈 배경에서 갑자기 떠오르는 느낌을 줄인다.
- UI smoke가 지면 밴드, 레인, 유닛 루트의 전장 y 범위를 검증한다.

## 비목표
- 카드 데이터, 군주, nation id, 세계관 정본은 변경하지 않는다.
- 전투 밸런스나 파도 구성은 변경하지 않는다.
- GUI PNG 실촬영 하네스는 별도 피처로 남긴다.

## 완료 기준
- `battle.gd` 배경 레이어에 `battlefield_floor_band` 1개 이상, `battlefield_depth_lane` 3개 이상이 생성된다.
- 배치 화면의 타일 접지 shadow와 지면 plate는 유지된다.
- 수동 첫 플레이 smoke에서 교전 phase 진입 후 필드 격자는 숨겨져 있고, 유닛 루트는 보드 지면 범위 안에 있다.
- `./init.sh`가 green이다.
