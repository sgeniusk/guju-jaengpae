# feat-058 다음 배치 손패 미리보기

## 목표
런맵과 전투 결과 화면이 현재 손패 수가 아니라 다음 전투 배치 단계에서 실제로 제시될 손패 후보 수를 기준으로 안내한다. 상점이나 전리품 뒤 카드가 늘어나도 다음 전투는 다시 손패 3장 중 1장 선택이라는 규칙이 흔들리지 않게 보인다.

## 범위
- `RunState`에 다음 배치 손패를 비파괴적으로 계산하는 preview API를 추가한다.
- `RunManager`가 preview와 준비 여부를 UI에 제공한다.
- `RunPrepSummary`는 preview 손패와 현재 손패 수를 함께 받아, 정리되는 경우 `다음 손패 X→3`을 표시한다.
- 전투 결과의 다음 준비 안내와 다음 스테이지 버튼 tooltip에 다음 배치 손패 안내를 추가한다.

## 비범위
- 손패 섞기, 카드 draw 순서, 보상 후보 정책, 카드 스키마는 변경하지 않는다.
- 전투 진입 시 실제 `prepare_deploy_hand()` 동작은 기존 그대로 유지한다.

## 검증
- 단위 테스트가 preview가 상태를 변형하지 않고 실제 prepare 결과와 일치하는지 검증한다.
- RunPrepSummary 테스트가 현재 손패와 preview 손패가 다를 때 정리 문구를 표시하는지 검증한다.
- UI smoke가 보상 후 다음 배치 손패 안내와 tooltip을 확인한다.
