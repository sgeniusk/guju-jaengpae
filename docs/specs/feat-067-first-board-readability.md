# feat-067 — 첫 전투 보드 가독성 polish

## 목표
첫 전투 배치 화면에서 플레이어가 보드 칸의 역할을 즉시 읽게 한다.

## 배경
현재 첫 전투는 성 위치 선택 → 손패 3장 중 1장 선택 → 빈 타일 배치 또는 계략/우물 사용 → 즉시 교전 구조다. 패널 안내와 tooltip은 있지만, 실제 보드 칸은 성 선택 전이나 손패 선택 전에는 빈 라벨로 남아 있어 “어디를 눌러야 하는지”가 약하게 보인다.

## 범위
- 성 선택 전 빈 타일에 `성 후보` visible label과 tooltip 표시
- 성 선택 후 손패 미선택 상태 빈 타일에 `손패 선택` visible label과 tooltip 표시
- 유닛/건물 선택 후 빈 타일에 `배치 가능` visible label과 카드명 포함 tooltip 표시
- 계략 선택 후 빈 타일에 `계략 버튼` visible label과 계략 버튼 사용 tooltip 표시
- UI smoke가 성 후보, 손패 선택, 계략 버튼, 배치 가능 상태를 검증

## 비범위
- 배치 규칙, 보드 크기, 성 위치 규칙, 전투 수치 변경
- 새 아트 에셋 추가
- 카드/지형/진영 데이터 변경

## 수용 기준
- 첫 전투 진입 직후 빈 타일에 `성 후보`가 보인다.
- 성 위치를 고르면 남은 빈 타일에 `손패 선택`이 보인다.
- 계략 카드를 선택하면 빈 타일이 `계략 버튼` 상태를 말한다.
- 병종/건물 카드를 선택하면 빈 타일이 `배치 가능` 상태를 말한다.
- `tools/ui_feedback_smoke.gd`, `./init.sh`가 통과한다.
