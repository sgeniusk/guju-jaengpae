# feat-045 — 집중표적 체감 피드백

## 목표
전투 중 집중표적 명령이 실제 지휘처럼 읽히게 한다.

## 범위
- 집중표적 버튼 tooltip이 전투 상태, 활성 여부, 현재 표적, 지휘 장수 수를 반영한다.
- 적 클릭 성공 시 힌트 문구에 표적명과 집중하는 장수 수를 표시한다.
- 지정 표적 위에 `집중` 라벨과 장수 수를 표시하고, 클릭 순간 표적까지 지휘선과 짧은 명령 배너를 띄운다.
- 빈 곳 클릭, 지휘할 장수 없음, 표적 사망 시 자동 표적으로 복귀했다는 문구를 남긴다.
- 표적 선택 반경과 문구는 순수 helper로 분리해 테스트한다.

## 비범위
- BattleSim 표적 우선순위 변경.
- 공격 속도, 피해량, 스킬 쿨다운 변경.
- 장수 개별 선택.
