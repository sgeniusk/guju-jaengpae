# feat-066 — 전투 유닛 접지감 보강

## 목표
전투 중 장수와 병사들이 공중에 떠 보이지 않고 지면 위에 서 있는 느낌을 강화한다.

## 배경
분대/호위 렌더는 병사 수를 늘려 삼국지 군세감을 만들었지만, root 단위 그림자만으로는 각 병사의 발밑 접지감이 약하다. 특히 장수 본체가 호위병 위에 살짝 떠 있는 구조는 화면에서 부자연스럽게 보일 수 있다.

## 범위
- 분대 병사와 장수 호위 각각의 발밑에 작은 ground shadow를 추가
- 장수 본체의 위쪽 오프셋을 줄이고 본체 발밑 shadow를 추가
- UI smoke에서 실제 전투 시작 후 ground shadow meta 노드가 생성되는지 검증

## 비범위
- 전투 수치, 이동, 충돌, 승패 조건 변경
- 새 아트 에셋 추가
- 카메라/배경/타일 구조 변경

## 수용 기준
- 첫 수동 전투 시작 후 유닛 visual tree에 ground shadow meta 노드가 충분히 존재한다.
- 분대/호위 병사의 shadow는 각 병사의 발 위치 근처에 붙는다.
- 장수 본체는 기존보다 덜 떠 보이도록 지면에 가깝게 내려온다.
- `tools/ui_feedback_smoke.gd`, `./init.sh`가 통과한다.
