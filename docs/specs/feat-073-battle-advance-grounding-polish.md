# feat-073 전투 진군 접지/충돌선 polish

## 목표
- 전투 시작 순간 병사들이 하늘이나 UI 위가 아니라 전장 바닥을 박차고 진군하는 것처럼 보인다.
- 양 진영이 중앙에서 부딪히는 지면 충돌선이 보여, 디펜스 게임의 적 출현이 아니라 양군 교전으로 읽힌다.
- 새 연출은 전투 수치와 승패를 바꾸지 않고, 자동 smoke가 VFX 계약을 검증한다.

## 범위
- `BattleFeel`에 전투 시작 지면 연출용 순수 마커를 추가한다.
- `battle.gd`가 마커를 바탕으로 진군 먼지와 지면 충돌선을 렌더링한다.
- `tools/ui_feedback_smoke.gd`가 첫 수동 전투에서 해당 VFX meta를 검증한다.
- `test/test_battle_feel.gd`가 마커 수, 좌우 진영, 레인 구성을 검증한다.

## 비범위
- 이동/공격/승패 판정 변경
- 새 아트 에셋 추가
- 카드, 진영, 지형 효과 밸런스 변경

## 수용 기준
- 첫 수동 전투 시작 후 `advance_dust` VFX가 양 진영 3레인에 충분히 생성된다.
- 첫 수동 전투 시작 후 `ground_clash` VFX가 3레인 중앙에 생성된다.
- `BattleFeel` 순수 helper 테스트와 UI smoke가 통과한다.
- `./init.sh`가 통과한다.
