# feat-075 전장 보드 지면화와 교전 가시성 보정

## 목표
- 배치 보드 9칸이 하늘에 뜬 UI 패널이 아니라 전장 바닥에 붙은 작전 표식으로 보인다.
- 배치 직후 교전이 시작될 때 유닛이 필드 뒤에서 나타나는 것처럼 보이지 않는다.
- 기존 성 선점, 손패 3장 중 1장, 즉시 교전 규칙과 전투 수치는 바꾸지 않는다.

## 범위
- 전투 화면 투영의 세로 위치와 압축 비율을 조정해 보드와 유닛 발 위치를 배경의 지면 영역으로 내린다.
- 타일 클릭/시각 diamond 크기를 실제 타일 텍스처 스케일과 맞추고, 타일별 접지 shadow를 추가한다.
- 교전 시작 후 배치용 격자는 빠르게 사라져 전투 유닛과 겹쳐 보이지 않게 한다.
- UI smoke가 보드 지면 밴드, 타일 접지 shadow, 교전 중 격자 숨김을 검증한다.

## 비범위
- 카드/군주/전투 수치 변경
- 새 배경 아트 추가
- 전투 AI, 보상, 저장, 맵 흐름 변경

## 수용 기준
- 첫 배치 보드 타일 중심의 상단 y가 화면 지면 밴드에 들어온다.
- 타일 세로 간격은 타일 높이와 맞아 9칸이 분리된 공중 패널처럼 보이지 않는다.
- 교전 phase 진입 후 `IsoBaseLayer`는 보이지 않아야 한다.
- `./init.sh`가 통과한다.
