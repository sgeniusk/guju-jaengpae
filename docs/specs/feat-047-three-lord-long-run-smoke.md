# feat-047 — 현세 3군주 장기런 스모크

## 목표
최종 보스까지 이어지는 장기런 검증이 유비 한 명이 아니라 현세 3군주 모두를 커버하게 한다.

## 범위
- `tools/long_run_smoke.gd`가 유비, 조조, 손권을 각각 새 `RunState`로 시작해 stage 15 최종 보스까지 검증한다.
- 군주별 전략 덱에 맞는 combat/shop priority를 둔다.
- 결정적 선택기는 유닛 배치와 증원뿐 아니라 건물 배치, 망루 오라, 계략 battle/run 효과, 보패 공격 보정을 실제 전투 입력에 반영한다.
- 각 군주는 성 위치 선택, 교전당 1장 플레이, stage 3/6/9/12 칙령, stage 4/8 상점, stage 11 사건, 보드 확장, 보패 공격 보정, 최종 보스 승리를 통과해야 한다.
- 출력은 군주별 구간과 최종 요약을 보여준다.
- 조조 첫 정예에서 드러난 원거리 스파이크는 `WaveFactory` 정예 명궁 수치를 좁게 낮춰 정상 3유닛+망루 루트가 막히지 않게 한다.
- 손권은 주유를 수군보다 먼저 세우는 장기런 우선순위로 오나라 화공 축을 검증한다.

## 비범위
- 카드 Resource 추가나 스키마 변경.
- 천계·마계 nation id 확장.
- 광범위한 밸런스 패스.
- UI 렌더 변경.
