# feat-042 — 진형 전술 시너지

## 목표
격자 배치가 단순 위치 선택이 아니라 전투 수치와 읽히는 전략으로 이어지게 한다. 새 카드 Resource 필드 없이 기존 장수·병종·행/열 정보만 사용한다.

## 범위
- 장수 인접 지휘 — 병종 카드가 상하좌우 인접 장수와 붙어 있으면 공격력 +10%.
- 전열 엄호 — 원거리 유닛 앞 같은 열에 근접 아군이 있으면 공격력 +15%.
- 측면 기동 — 기병이 좌우 가장자리 열에 배치되면 공격력 +10%.
- 전투 배치 타일 라벨에 적용된 전술 태그를 짧게 표시한다.

## 비범위
- 새 카드 스키마 필드 추가.
- 천계·마계 정본 확장.
- 전투 AI/타겟팅 규칙 변경.

## 검증
- 순수 helper 단위 테스트로 지휘·엄호·측면·비적용 케이스를 잠근다.
- `CardCatalog.build_board_army()` 경로에서 실제 공격 수치가 보정되는지 검증한다.
- `./init.sh` 전체 검증으로 기존 MVP 루프와 스모크를 재확인한다.
