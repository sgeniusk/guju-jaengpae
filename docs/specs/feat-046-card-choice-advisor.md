# feat-046 — 카드 선택 전략 안내

## 목표
상점과 전리품 선택이 단순 카드 나열이 아니라 현재 보드와 손패 기준의 전략적 선택으로 읽히게 한다.

## 범위
- 현재 보드, 보드 레벨, 손패, 골드, 카드 타입만 읽는 `CardChoiceAdvisor` 순수 helper를 추가한다.
- 같은 유닛 카드가 보드에 있으면 `증원 후보`와 다음 레벨을 표시한다.
- 장수, 병종, 건물, 계략, 보패의 역할을 `지휘 핵심`, `전열 보강`, `경제 확장`, `화력 거점`, `즉시 한 수`, `지속 화력`처럼 player-facing 문구로 설명한다.
- 상점에서는 골드가 부족한 카드에 `자금 부족`과 부족 금액을 표시한다.
- run_map 상점 카드와 battle 전리품 버튼의 visible text와 tooltip에 추천 문구를 추가한다.
- UI smoke가 상점과 전리품 화면에서 추천 문구가 실제로 렌더되는지 검증한다.

## 비범위
- RewardPool 확률이나 후보 선정 정책 변경.
- 카드 Resource 스키마 필드 추가.
- 저장 payload 변경.
- 밸런스 수치, 전투 판정, 상점 가격 변경.
