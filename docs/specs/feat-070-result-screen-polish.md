# feat-070 — 결과 화면 시각 polish

## 목표
전투 종료 오버레이의 첫 시선에서 결과, 런 상태, 다음 행동을 즉시 읽을 수 있게 한다.

## 배경
현재 결과 오버레이는 프로필 기록, 런 계속/종료 안내, 결산, 보상 또는 새 런 버튼을 순서대로 보여준다. 정보는 충분하지만 화면 첫머리에 명확한 결과 배너가 없어 패배/최종승리/일반승리의 다음 행동이 늦게 읽힌다.

## 범위
- `BattleOutcomeGuide`에 결과 배너 제목, 상세, 다음 행동 문구를 추가한다.
- `battle.gd` 결과 오버레이 최상단에 배너 문구를 배치한다.
- 패배와 최종승리는 새 런 경로를, 일반 승리는 전리품 선택 후 런맵 복귀 경로를 분명히 표시한다.
- 기존 런 결산, 보상 선택, 다음 스테이지 버튼 동작은 유지한다.

## 비범위
- 전투 시뮬레이션, 보상 후보, 런 저장 로직 변경
- 신규 이미지 에셋, 사운드, 애니메이션 추가
- 결과 화면 레이아웃 전체 재설계

## 수용 기준
- 패배 결과 화면에 `결과 — 성 함락`과 `다음 행동 — 군주 선택으로 새 런`이 보인다.
- 최종승리 결과 화면에 `결과 — 구주 정복`과 `다음 행동 — 군주 선택으로 새 런`이 보인다.
- 일반 승리 보상 화면에 `결과 — 전투 승리`와 `다음 행동 — 전리품 선택 후 런맵 복귀`가 보인다.
- `tools/battle_result_smoke.gd`, `tools/ui_feedback_smoke.gd`, `./init.sh`가 통과한다.
