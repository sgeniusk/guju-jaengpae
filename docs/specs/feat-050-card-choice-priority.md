# feat-050 — 카드 선택 추천순 정렬

## 목표
상점과 전리품 선택지가 단순 나열이 아니라 현재 런 맥락에서 더 좋은 선택부터 보이게 한다.

## 범위
- `CardChoiceAdvisor`에 후보 id 배열을 추천 점수순으로 정렬하는 순수 helper를 추가한다.
- 동일 점수는 기존 후보 순서를 유지해 보상 후보의 랜덤성과 결정적 tie-break를 보존한다.
- 상점 카드 렌더 순서를 추천순으로 바꾼다. 골드가 부족한 카드는 낮은 점수로 뒤로 밀린다.
- 전리품 카드 버튼도 같은 추천순으로 렌더한다.
- UI smoke가 상점에서 증원 후보가 첫 추천 카드로 올라오는지 확인한다.

## 비범위
- RewardPool 후보 생성, 확률, unlock 정책 변경.
- 카드 Resource 필드 추가.
- 상점 가격, 골드 지급, 보상 수량 변경.
- 새 카드 또는 새 진영 추가.
