# feat-062 런맵 진행 리듬 안내

## 목표
런맵에서 플레이어가 현재 스테이지의 의미와 앞으로 이어질 전투, 칙령, 상점, 보스 흐름을 한눈에 이해하게 한다.

## 범위
- 현재 스테이지의 행동을 `진행 리듬` 문구로 표시한다.
- 현재 스테이지 뒤 3개 스테이지의 종류를 짧은 줄로 보여준다.
- 문구 계산은 `RunFlowSummary` 순수 helper로 분리해 테스트한다.
- 런맵의 전투, 칙령, 상점, 사건 화면 모두 같은 진행 리듬 안내를 표시한다.

## 비범위
- StageCadence의 실제 간격, 난이도, 보상, 전투 결과, 저장 로직은 변경하지 않는다.
- 새 맵 분기, 장기런 밸런스, 카드 데이터 추가는 이번 피처에 포함하지 않는다.
- 그래픽 배경이나 대규모 레이아웃 개편은 하지 않는다.

## 검증
- 단위 테스트가 현재 스테이지 행동, 다음 3칸 요약, 최종 보스 표현을 검증한다.
- UI smoke가 런맵 첫 전투와 상점 화면에서 진행 리듬 문구와 다음 스테이지 안내를 확인한다.
- `./init.sh` 전체 검증을 통과한다.
