# feat-082 — 전장 footline/depth 재수정

## 문제

사용자 플레이 피드백에서 배치 필드 9칸이 여전히 공중 UI처럼 보이고, 성/유닛이 필드 뒤에서 나타나는 느낌이 남았다. 기존 feat-081은 보드 y와 alpha를 낮췄지만, 유닛 발 위치가 타일 다이아몬드 하단보다 충분히 앞쪽에 있지 않아 격자 안쪽에 서 있는 착시가 남았다.

## 목표

- 배치 보드 중심을 조금 더 전경 바닥 쪽으로 내린다.
- 유닛, 성, 건물, 지면 VFX의 foot 위치를 타일 다이아몬드 하단보다 앞쪽에 둔다.
- 성/배치된 카드 타일의 fill/outline을 더 낮춰 유닛 주변에 빛나는 판이 남지 않게 한다.
- 교전 중에는 기존처럼 배치 격자를 즉시 숨겨 실제 싸움에 격자가 끼어들지 않게 한다.

## 비범위

- 카드/전투 밸런스 변경 없음.
- 배치 규칙, 손패, 성 위치 선택 규칙 변경 없음.
- 배경 에셋 교체 없음.

## 완료 조건

- `test/test_unit_walk_visuals.gd`가 보드 y 범위와 유닛 footline이 타일 하단을 넘는 조건을 검증한다.
- `tools/ui_feedback_smoke.gd`가 배치 및 교전 depth 회귀를 검증한다.
- `./init.sh`가 green이다.
