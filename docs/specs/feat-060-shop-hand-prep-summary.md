# feat-060 상점 손패 정리 안내

## 목표
상점에서 카드를 구매한 뒤 현재 손패가 늘어나도 다음 전투는 다시 배치 후보 3장 중 1장을 쓰는 구조를 상점 화면에서 바로 읽을 수 있게 한다.

## 범위
- 상점 패널에 `다음 전투 손패 — 후보 N장 중 1장` 요약을 표시한다.
- 현재 상점 손패 수와 다음 전투 배치 후보 수가 다르면 `상점 손패 X장 → 전투 후보 Y장`으로 정리 흐름을 보여준다.
- tooltip은 구매 카드가 현재 손패에 들어온 뒤 전투 진입 시 드로우 더미로 돌아가고 다음 후보가 열린다는 점을 설명한다.
- 문구 계산은 `ShopHandSummary` 순수 helper로 분리해 테스트한다.

## 비범위
- 상점 판매 목록, 구매 가격, 보상 후보, draw 순서, `prepare_deploy_hand()` 동작은 변경하지 않는다.
- 다중 구매 제한이나 구매 후 즉시 배치 기능은 이번 피처에 포함하지 않는다.

## 검증
- 단위 테스트가 상점 손패 5장 → 전투 후보 3장 문구와 이미 준비된 후보 문구를 검증한다.
- UI smoke가 상점 화면에서 다음 전투 손패 요약과 드로우 더미 tooltip을 확인한다.
- `./init.sh` 전체 검증을 통과한다.
