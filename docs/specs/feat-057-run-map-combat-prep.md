# feat-057 런맵 전투 준비 패널 강화

## 목표
전투 진입 직전 런맵에서 이번 교전의 준비 상태를 읽을 수 있게 한다. 플레이어는 전투 화면에 들어가기 전에 성 위치 선택 여부, 손패 3장 중 1장 규칙, 보드 군세, 증원/배치/계략 후보 수를 확인한다.

## 범위
- combat, elite, boss 스테이지의 전투 시작 버튼 위에 준비 요약을 표시한다.
- 준비 요약 계산은 `RunPrepSummary` 순수 helper로 분리한다.
- helper는 현재 보드, 보드 레벨, 손패, 성 위치, 보드 용량, 카드 catalog만 사용한다.
- 전투 로직, 카드 스키마, nation id, 밸런스 수치는 변경하지 않는다.

## UI 계약
- 제목 문구는 `전투 준비 — 손패 3장 중 1장` 형태를 유지한다.
- 상세 문구는 `성 위치`, `군세`, `증원 후보`, `배치 후보`, `계략`을 포함한다.
- tooltip은 전투 화면에서 성 위치를 고르고 손패 한 장만 사용한 뒤 즉시 교전한다는 루프를 설명한다.

## 검증
- 단위 테스트가 빈 보드 첫 손패와 기존 부대 증원 손패를 검증한다.
- UI smoke가 첫 런맵 전투 화면에 준비 요약과 tooltip이 렌더되는지 검증한다.
- `./init.sh` 전체 검증이 green이어야 한다.
