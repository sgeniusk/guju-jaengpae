# 구주쟁패 (九州爭霸) — 게임 컨셉 시드

새 세션의 하네스 구성이 소비할 기획 시드다. 완성 설계가 아니라 출발점이다. `harness-start` 스킬이 이 문서를 읽어 v0.1 정의를 채운다.

## 한 줄 정체성
삼국지 기반에 판타지를 얹은 **덱빌딩 로그라이크 전쟁 게임**. *Nine Kings*를 벤치마킹한다.

## 벤치마크 — Nine Kings에서 가져올 것
- 매 런마다 카드(왕/유닛)를 배치해 파도를 막아내는 **덱빌딩 + 타워/전열 방어**의 결합.
- 런을 반복하며 새 왕·카드를 **영구 해금**하는 메타 progression.
- 짧고 밀도 높은 한 판, 다음 판이 또 다른 빌드를 강제하는 리플레이성.

## 핵심 루프 (가설 — 새 세션에서 확정)
1. **군주(君主) 선택** — 조조·유비·손권 등. 군주마다 시작 덱·특성·판타지 색채가 다르다.
2. **전투** — 로그라이크식 한 판. 장수·병종 카드를 배치해 적 파도를 막거나 적진을 친다.
3. **전리(戰利)** — 승리 시 새 장수 카드·병종·유물(보패)을 골라 덱에 편입.
4. **반복·해금** — 런이 끝나면 장수·병종이 영구 해금되어 다음 런의 선택지가 넓어진다.

## 빌딩 블록
- **장수(將帥)** — 카드의 핵심. 관우·장비·제갈량 등. 고유 스킬·병종 시너지.
- **병종(兵種)** — 보병·궁병·기병·노병·수군 + 판타지 병종(아래).
- **덱빌딩** — 장수·병종·계략·보패 카드를 조합해 빌드를 만든다.
- **로그라이크** — 매 런 무작위 노드·이벤트·보상, 영구사 + 영구 해금.

## 판타지 레이어 (삼국지 + 환상)
- **선술(仙術)·기문둔갑** — 제갈량·좌자 등 도사 계열이 마법처럼 작동.
- **요괴·신화화된 적** — 황건적을 사령(死靈)·요괴 군세로, 동탁을 마왕급 보스로.
- **신격화된 장수** — 관우=무신(武神), 여포=귀병(鬼兵)의 화신 같은 과장.
- **보패(寶貝)** — 유물 시스템을 신화 아이템(방천화극·청룡언월도에 신력 부여)으로.
톤은 정사·연의 + 봉신연의(封神演義)풍 환상의 교차.

## v0.1 성공 기준 (초안 — 새 세션에서 확정)
한 판의 전투 루프가 돈다 — 군주 하나, 장수 3종·병종 3종으로 적 파도 한 번을 막고, 승리 시 카드 하나를 보상으로 덱에 넣는 최소 플레이 가능 빌드.
